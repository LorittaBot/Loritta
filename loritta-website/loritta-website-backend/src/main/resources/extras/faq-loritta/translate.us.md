## Yay! I helped translate, but... My name doesn't appear on the translator list.

Don't get discouraged, young grasshopper! There is a very easy method to add your name there. Just make a [Pull Request](https://docs.github.com/en/pull-requests/collaborating-with-pull-requests/proposing-changes-to-your-work-with-pull-requests/creating-a-pull-request) on our [locales repository](https://github.com/LorittaBot/LorittaLocales), adding your Discord ID to the list in the `/{language-folder}/translators.yml` file (check the default folder for an example!).

In the Pull Request, add a link to your Crowdin profile. Credited users are shown in the `+language` command and receive a special role on Loritta's support server!

Remember that we will only accept it if we think you really helped and if your contribution was substantial! You won't be accepted if you only help with a few translations, and if necessary, we might remove many of your translations (please, do not use Google Translate, we know if you used it).
